# 마음 한 조각 💌 — 익명 고민·칭찬 게시판

**Server + DB (Supabase)** 를 활용한 익명 게시판입니다.
누구나 익명으로 글(고민 💭 / 칭찬 👏 / 응원 💪)을 남기고, 서로의 글에 **공감**을 눌러줄 수 있어요.

> 핵심 학습 포인트: 공감 버튼을 누를 때마다 DB의 `likes` 값이 **UPDATE** 되면서 숫자가 올라갑니다.
> "DB에 저장된 데이터를 수정한다"는 개념을 눈으로 직접 확인하는 게 목표예요.

---

## 🧱 구조

```
[글 작성 (카테고리 + 내용)]
        │  insert
        ▼
   ┌──────────────┐        select + order        ┌──────────────┐
   │   Supabase    │  ◀───────────────────────▶  │   화면(목록)   │
   │  (Server+DB)  │                              │  최신순/공감순  │
   └──────────────┘        update (likes+1)       └──────────────┘
        ▲                                                 │
        └──────────────── 공감 버튼 클릭 ◀─────────────────┘
```

- **DB 역할:** 게시글 저장 · 공감 수 관리 · 정렬/필터 조회
- **Server 역할:** Supabase가 자동 생성하는 API가 곧 서버입니다 (별도 Node 서버 없음)

이 앱은 `index.html` **단 한 개 파일**로 동작합니다. 브라우저에서 Supabase JS 클라이언트(CDN)로
DB에 직접 읽고/쓰기 때문에, Supabase 프로젝트 하나만 있으면 됩니다.

---

## 🚀 시작하기 (3단계)

### 1. Supabase 프로젝트 만들기
1. [supabase.com](https://supabase.com) 가입 (무료)
2. **New project** → 이름/비밀번호 입력 후 생성 (1~2분 소요)

### 2. 테이블 만들기
1. 왼쪽 메뉴 **SQL Editor** → **New query**
2. 같은 폴더의 [`supabase-schema.sql`](./supabase-schema.sql) 내용을 통째로 붙여넣고 **RUN(▶)**
   - `posts` 테이블 + 익명 읽기/쓰기/수정 권한(RLS)이 한 번에 설정됩니다.

### 3. 앱 실행 + 연결
1. 이 폴더에서 로컬 서버 실행:
   ```bash
   npx serve .
   ```
   또는 VS Code의 **Live Server** 확장으로 `index.html` 열기
2. 처음 열면 **설정 화면**이 나옵니다. Supabase 대시보드의
   **Project Settings → API** 에서 아래 두 값을 복사해 붙여넣으세요:
   - **Project URL**
   - **anon public** key  *(공개되어도 안전한 키예요 — 권한은 위에서 만든 RLS가 지켜줍니다)*
3. **저장하고 시작하기** → 게시판 완성! 🎉

> 입력한 값은 브라우저 `localStorage`에 저장되어, 다음에 열 때 자동으로 연결됩니다.

## ✨ 기능

| 기능 | 동작 | Supabase 호출 |
|------|------|----------------|
| 익명 글 작성 | 카테고리 선택 + 내용 입력 (작성자 정보 없음) | `insert({ category, content })` |
| 목록 조회 | 카드 형태로 표시 + 카테고리 필터 | `select('*').eq('category', …)` |
| **공감 버튼** | 클릭 시 공감 수 **+1** | `update({ likes }).eq('id', …)` |
| 정렬 | **최신순** / **공감순** 토글 | `.order('created_at' / 'likes', …)` |

---

## 🗄️ posts 테이블

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `id` | bigint | 글 고유 번호 (자동 증가) |
| `category` | text | `고민` / `칭찬` / `응원` |
| `content` | text | 글 내용 |
| `likes` | int | 공감 수 (기본값 0) |
| `created_at` | timestamptz | 작성 시각 (기본값 now) |

## 🛟 문제 해결
- **"권한이 없어요 / RLS" 오류** → 2단계 SQL을 다시 실행했는지 확인하세요 (정책 생성 포함).
- **글이 안 보여요** → Project URL과 anon key를 정확히 붙여넣었는지, 설정 화면을 다시 열어 확인하세요.
- **설정을 바꾸고 싶어요** → 브라우저 콘솔에서 `localStorage.clear()` 후 새로고침하면 설정 화면이 다시 나옵니다.
